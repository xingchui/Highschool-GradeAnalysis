关于系统

高中成绩分析系统

版本: 2.0.0

这是一个基于Python Flask框架开发的高中成绩分析系统，旨在帮助教师更好地分析学生成绩数据，提供全方位的统计分析和趋势追踪功能。

主要功能
  • Excel成绩文件上传和解析

    支持多种Excel格式，自动识别数据结构

  • 学生排名

    全校排名和班级排名，支持分页查看

  • 985/211/一本分数线统计分析

    按班级、科目统计上线人数和上线率

  • 单科分数线设置

    支持语数外150分制及各科目自定义分数线

  • 学生成绩趋势分析

    对比多次考试成绩，追踪学生进步情况

技术栈
后端
  • Python 3.10+
  • Flask 3.x
  • Pandas
前端
  • Bootstrap 5
  • Plotly.js
  • Bootstrap Icons
